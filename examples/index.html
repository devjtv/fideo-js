Fideo JS

Normal video and iframe embeds with a small custom UI, responsive sources, viewport playback, and per-control configuration.

HTML5 video, full controls, viewport play/pause

<video
  data-fideo
  data-fideo-muted="true"
  data-fideo-viewport="play-pause"
  data-fideo-src="/videos/desktop.mp4"
  data-fideo-src-mobile="/videos/mobile.mp4"
  data-fideo-poster="/posters/desktop.jpg"
></video>

Minimal controls: play and timeline only

<video
  data-fideo
  data-fideo-show-volume="false"
  data-fideo-show-settings="false"
  data-fideo-show-fullscreen="false"
  data-fideo-src="/videos/clip.mp4"
></video>

Timeline only, no buttons

<video
  data-fideo
  data-fideo-show-play="false"
  data-fideo-show-time="false"
  data-fideo-show-volume="false"
  data-fideo-show-settings="false"
  data-fideo-show-fullscreen="false"
  data-fideo-src="/videos/quiet-loop.mp4"
></video>

YouTube no-cookie embed

<iframe
  data-fideo
  data-fideo-muted="true"
  src="https://www.youtube-nocookie.com/embed/M7lc1UVf-VE"

Looping autoplay, no controls

<video
  data-fideo
  data-fideo-autoplay="true"
  data-fideo-muted="true"
  data-fideo-loop="true"
  data-fideo-controls="false"
  data-fideo-src="/videos/background-loop.mp4"
  data-fideo-poster="/posters/background-loop.jpg"
></video>

JavaScript object initialization

import { Fideo } from 'fideo-js';

new Fideo('#object-player', {
  autoplay: true,
  muted: true,
  loop: true,
  controls: false,
  sources: {
    desktop: '/videos/background-loop.mp4',
  },
  posters: {
    desktop: '/posters/background-loop.jpg',
  },
});

Background video hero

Full bleed, no controls

Background mode fills the parent container, mutes, loops, autoplays, and keeps controls hidden.

<div class="hero">
  <video
    data-fideo
    data-fideo-background="true"
    data-fideo-src="/videos/hero.mp4"
    data-fideo-poster="/posters/hero.jpg"
  ></video>
</div>

Background iframe banner with cover sizing

Iframe cover

Iframe media is resized by Fideo so it covers unusual container shapes without black bars.

<div class="banner">
  <iframe
    data-fideo
    data-fideo-background="true"
    data-fideo-background-aspect-ratio="16:9"
    src="https://vimeo.com/76979871"
  ></iframe>
</div>

Full browser width Vimeo iframe hero

Vimeo cover hero

A fixed-height, full viewport-width hero using an iframe background with cover sizing.

<section class="hero">
  <iframe
    data-fideo
    data-fideo-background="true"
    data-fideo-background-aspect-ratio="16:9"
    src="https://vimeo.com/76979871"
  ></iframe>
</section>

Full browser width YouTube iframe hero

YouTube cover hero

<section class="hero">

    data-fideo
    data-fideo-background="true"
    data-fideo-background-aspect-ratio="16:9"
    src="https://www.youtube.com/watch?v=M7lc1UVf-VE"
  ></iframe>
</section>
