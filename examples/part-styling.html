::part() Styling Demo

Each example below styles Fideo controls exclusively through CSS ::part() selectors — no custom element leaking or !important battles.

part="play-button"

Recolor the play button and enlarge its SVG icon.

.fideo__controls::part(play-button) {
  color: #ff6b6b;
}
.fideo__controls::part(play-button) svg {
  width: 22px;
  height: 22px;
}

part="timeline"

Thicken the track, always-show the thumb, and use a larger accent-colored knob.

.fideo__controls::part(timeline) {
  --fideo-track-size: 8px;
}
.fideo__controls::part(timeline)::-webkit-slider-thumb {
  width: 18px;
  height: 18px;
  background: #46d9a7;
  opacity: 1;
}

part="mute-button"

Give the mute toggle a circular background with a tinted hover state.

.fideo__controls::part(mute-button) {
  border-radius: 50%;
  background: rgba(255,255,255,0.06);
}
.fideo__controls::part(mute-button):hover {
  background: rgba(70,217,167,0.15);
}

part="fullscreen-button"

Set a custom brand-color for the fullscreen toggle.

.fideo__controls::part(fullscreen-button) {
  color: #f9ca24;
}
.fideo__controls::part(fullscreen-button):hover {
  color: #fbd440;
}

part="current-time" & part="duration"

Change time-display color and bump up the font size.

.fideo__controls::part(current-time),
.fideo__controls::part(duration) {
  color: #46d9a7;
  font-size: 14px;
}

part="volume-slider"

Widen the vertical volume track and give the thumb a punchy accent.

.fideo__controls::part(volume-slider) {
  --fideo-track-size: 6px;
}
.fideo__controls::part(volume-slider)::-webkit-slider-thumb {
  width: 16px;
  height: 16px;
  background: #ff6b6b;
  margin-left: -5px;
}

part="settings-button"

Turn the settings gear into a pill-shaped affordance.

.fideo__controls::part(settings-button) {
  border-radius: 20px;
  padding: 0 10px;
}

part="speed-button"

Give speed-rate menu items a softer border and tinted hover.

.fideo__controls::part(speed-button) {
  border-radius: 4px;
}
.fideo__controls::part(speed-button):hover {
  background: rgba(70,217,167,0.18);
}
